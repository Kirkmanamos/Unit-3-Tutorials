# Unit 3 - Rational Functions Tutorials

This repository contains interactive tutorials and tools for Precalculus Unit 3, focusing on Rational Functions.

## Table of Contents

### Core Concepts

- **[Rational Features](RationalFeatures.html)**: Analyze rational functions to find their domain, holes, vertical asymptotes, and intercepts.
- **[Graphing Rational Functions](RationalGraphing.html)**: Interactive tool for visualizing rational function graphs and their key features.
- **[Linear Slope Explorer](LinearSlope.html)**: A simple explorer for linear functions (`f(x) = mx`) to understand slope and linearity.

### Partial Fraction Decomposition

- **[Partial Fractions (Basic)](PartialFractions.html)**: Introduction to partial fraction decomposition with standard linear factors.
- **[Repeated Linear Factors](PartialFractions_Repeated.html)**: Learn how to handle rational expressions with repeated linear factors in the denominator.
- **[Improper Rational Expressions](PartialFractions_Improper.html)**: Techniques for decomposing improper rational expressions (where the degree of the numerator is greater than or equal to the degree of the denominator).

### Navigation

- **[Main Index](index.html)**: The central landing page for all Unit 3 tutorials.
